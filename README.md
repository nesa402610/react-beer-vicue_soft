# Пивной поиск
Поиск по пиву. Вводим название и получаем различные виды пива. 
Тестовое задание. 

Сделано в точности по ТЗ.

## Стек
* ReactJS - как база приложения. 
• Axios - для работы с API. 

## Функционал:
* Поиск
* Своя страница для объекта
* Пагинация
